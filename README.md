An advanced **Data Analysis & Machine Learning dashboard** built to analyze the Netflix Movies and TV Shows dataset.

The project performs **Exploratory Data Analysis (EDA), interactive visualizations, machine learning clustering, and a recommendation engine** to uncover insights about Netflix content trends.

---

# 📊 Project Overview

This project explores the Netflix dataset to understand:

• Content growth trends  
• Global distribution of Netflix titles  
• Popular genres and content types  
• Director and actor contributions  
• Content duration strategies  
• Machine learning based content clustering  
• Content recommendation system  

The dashboard allows users to **interactively explore Netflix data through visual analytics**.

---

# 🚀 Key Features

### 📈 Content Trends Analysis
- Netflix content growth over time
- Movies vs TV Shows distribution
- Yearly content release trends

### 🌍 Global Content Distribution
- Top countries producing Netflix content
- Country share visualization
- Interactive world map showing global Netflix content distribution

### 🎭 Genre Insights
- Most popular genres
- Genre evolution across years
- Genre distribution patterns

### ⭐ Cast & Crew Analysis
- Top directors on Netflix
- Most frequently appearing actors
- Content duration distribution

### 📊 Machine Learning – Content Clustering
Using **K-Means Clustering** to group Netflix titles based on:

- Genre
- Description
- Content similarity

### 🤖 Recommendation Engine
A **content-based recommendation system** built using:

- TF-IDF Vectorization
- Cosine Similarity

Users can select a title and receive **similar content recommendations**.

### 📊 Strategy Insights
- Popularity score analysis
- Duration strategy for movies vs TV shows
- Netflix content growth forecasting using **Linear Regression**

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Vasantlohar0504/netflix-data-analysis-dashboard.git
````

Navigate to project folder

```bash
cd netflix-data-analysis-dashboard
```

Install required dependencies

```bash
pip install -r requirements.txt
```

Run the Streamlit dashboard

```bash
streamlit run dashboard/app.py
```
